# 🐄 QurbaniHat – Livestock Booking Platform

## 🌐 Live Site
https://qurbani-hat-delta.vercel.app

---

## 📌 Project Purpose
QurbaniHat is a modern livestock marketplace where users can explore animals (cows & goats) for Qurbani. Users can view detailed information and book animals securely after authentication.

---

## ✨ Key Features

- 🔐 Better-Auth Authentication (Login, Register, Google Login)
- 🐄 Browse All Animals with detailed information
- 📄 Protected Animal Details Page
- 📝 Booking Form with success toast (no DB storage)
- 👤 User Profile Page (Name, Email, Photo)
- ✏️ Update Profile Feature
- 🎬 Animate.css animation
- 🔔 Toast Notifications
- ⏳ Loading Spinner / Skeleton UI
- 🚫 Custom 404 Not Found Page
- 📱 Fully Responsive (Mobile, Tablet, Desktop)

---

## 🧰 Technologies Used

- Next.js (App Router)
- React.js
- Tailwind CSS
- HeroUI
- Better-Auth Authentication
